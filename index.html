Visual Lego
C'est Quoi?

Accueillez la toute nouvelle fonctionnalité de Lego avec la puissance de Sony.
Le fruit de cette collaboration est le tout nouveau Visual Lego, le premier ecran dans l’univers de Lego qui s’emporte partout autour de vous.

Visual Lego
Comment ça marche

Les briques de Lego sont recouvertes de pixels microLED avec une technologie de capteurs interconnectés, chaque brique est un écran qui est modulable grâce à l’ajout de briques .
Pour regarder n’importe quel flux il suffit de se connecter à son ensemble visual lego grâce à une application.

UNE MARQUE RESPONSABLE

LES BRIQUES SONT CONÇUES DANS LE RESPECT DU BIEN-ETRE DE L’ENVIRONNEMENT tout en gardant leurs solidités.
